How to use cpanel or shell to send?
1.) Upload send.php to the respective cpanel/shell
2.) now go to your_host_url/send.php to see if it's a compatible host
3.) copy the link given in the send.php and put in "php_c2_server_url" in config
4.) now you can send via shell without smtp on the sender
